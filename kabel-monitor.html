Kabel Monitor

Sistem DVOR - Politeknik Penerbangan Medan

Nama Kabel Kabel Monitor
Fungsi memonitor sinyal yang dipancarkan dan mendeteksi kesalahan.Ketika kontrol level signal tidak terpancar, akan memberikan peringatan dan memindahkan ke pemancara standby untuk melakukan pemulihan.
Input Sinyal hasil monitoring dari antena / monitor unit
Output Perangkat monitor DVOR / panel monitoring
Frekuensi Mengikuti frekuensi kerja DVOR, yaitu 108.00 – 117.95 MHz
Keterangan Kabel monitor digunakan untuk memastikan sinyal DVOR tetap sesuai parameter dan aman digunakan sebagai alat bantu navigasi.
